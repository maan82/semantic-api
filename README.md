 README
========

## Requisites
- Java
- Gradle
- Preferred IDE/Text editor
- Internet connection

## Basic commands
- `$ ./gradlew test`: run tests
- `$ ./gradlew build`: builds the project
- `$ java -jar build/libs/semantic-api-0.1.0.jar`: runs the server

Point your browser to [http://localhost:8080/](http://localhost:8080/) to get api home page.
